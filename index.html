Platform Overview

Mission

Automate ISO 27001 compliance for startups and enterprises with human-in-the-loop workflows, continuous monitoring, and audit-ready evidence collection. Built with FastAPI, LangGraph, and Next.js.

Architecture

  • • Backend: FastAPI + LangGraph + LCEL
  • • Frontend: Next.js 14 + React + Tailwind CSS
  • • AI Workflows: LangGraph with approval gates
  • • Database: SQLModel + PostgreSQL
